# Sample assets — مجلد العينات

ضع الملفات التالية هنا بالأسماء المحددة بالضبط:

| اسم الملف | المحتوى |
|---|---|
| `video.mp4` | الفيديو المخرج |
| `poster.png` | صورة البوستر العلمي |
| `podcast.mp3` | البودكاست |
| `slides.pdf` | PDF يُعرض كأنه عرض تقديمي |
| `paper.pdf` | (اختياري) الورقة البحثية الأصلية للمعاينة |

التطبيق يعمل بدون هذه الملفات (لكن العرض سيكون بلا وسائط حقيقية).
